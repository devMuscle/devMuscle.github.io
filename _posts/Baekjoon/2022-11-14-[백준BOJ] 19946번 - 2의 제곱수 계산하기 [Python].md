**문제 링크** : <https://www.acmicpc.net/problem/19946>

## 풀이 방법

2의 제곱수에서 -1을 하게 되면 홀수가 될 것이기 때문에 2로 나누어 떨어지지 않을것이다.<br>
이 부분을 활용해서 문제를 해결하면 된다.

## 새로 알게된 점

파이썬은 나눗셈의 몫을 구하고 싶으면 '//' 연사자를 사용해야 한다.

```python
n = int(input())

result = 64

while n > 0 :
    if n%2==0 :
        result-=1
        n//=2
    else :
        break

print(result)
```
